# 📸 Human Authenticity Source Folder (Effort Premium)

감독님, 이 폴더는 유튜브 2026 트렌드인 **'노력 프리미엄'**과 **'진정성'**을 확보하기 위한 핵심 창구입니다.

### 🛠️ 사용 방법
1.  **진짜 소스 넣기**: 감독님이 직접 찍은 사진(편의점 방문, 제품 구매, 영수증, 길거리 풍경 등)을 이 폴더에 넣어주세요. (JPG, PNG 형식)
2.  **자동 합성**: 영상을 생성할 때, 시스템이 이 폴더에서 이미지를 무작위로 하나 골라 영상 마지막에 **"실증 데이터 확인됨"** 이라는 인증 마크와 함께 3.5초간 노출합니다.
3.  **효과**: 시청자들은 100% AI 영상이 아닌, 사람이 직접 발로 뛴 '진짜 정보'라는 신뢰를 갖게 되어 시청 지속시간(Retention)이 상승합니다.

### 💡 팁
*   고화질이 아니어도 좋습니다. 오히려 스마트폰으로 대충 찍은 느낌이 더 '진정성' 있게 느껴질 수 있습니다.
*   자주 사진을 교체해 주시면 영상마다 다른 인증샷이 들어가서 채널의 생동감이 살아납니다.
