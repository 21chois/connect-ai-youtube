# 🏢 Connect AI (AI 1매뉴얼인 기업) 시스템 운영 

본 문서는 **조감독 영식(Agent Young-sik)**이 관리하는 유튜브 자동화 시스템의 주요 실행 순서와 방법을 정리한 공식 가이드입니다.

---

## 🚀 1. 시스템 핵심 파이프라인 (Total Automation)

가장 권장되는 실행 방식으로, 트렌드 분석부터 성과 피드백까지 모든 과정을 스스로 수행합니다.

### 📋 실행 순서
1.  **터미널 열기:** 프로젝트 루트 폴더에서 터미널을 실행합니다.
2.  **명령어 입력:**
    ```bash
    python main_orchestrator.py
    ```
3.  **내부 로직:**
    -   **Trend Analysis:** 현재 한국에서 가장 핫한 커머스 아이템 발굴
    -   **Video Gen:** Veo 3.1을 활용한 고화질 영상 생성
    -   **SEO Upload:** 알고리즘 최적화 제목/태그로 유튜브 업로드
    -   **RL Feedback:** 24시간 후 성과 평가 및 학습 데이터 저장

---

## 🌙 2. 야간 집중 운영 모드 (Night Mode)

대표님이 주무시는 동안 최대 3편의 영상을 최적의 간격으로 업로드합니다.

### 📋 실행 순서
1.  **명령어 입력:**
    ```bash
    python night_orchestrator.py
    ```
2.  **운영 타임라인:**
    -   **T+0h:** `냥도사 V3` (로또/감성) 업로드
    -   **T+2h:** `커머스 트렌드 1편` 업로드
    -   **T+4h:** `커머스 트렌드 2편` 업로드

---

## 📅 3. 데일리 시리즈 루프 (Daily Series Loop)

정해진 시간(12:00, 19:00)에 맞춰 매일 반복적으로 콘텐츠를 연재합니다.

### 📋 실행 순서
1.  **명령어 입력:**
    ```bash
    python run_dubai_chocolate_daily.py
    ```
2.  **연재 스케줄:**
    -   **12:00:** 메인 채널 (두바이 초콜릿 시리즈)
    -   **19:00:** 서브 채널 (영험한 냥도사 로또)

---

## 🛠️ 4. 개별 콘텐츠 단독 실행

특정 테마의 영상을 즉시 제작하고 싶을 때 사용합니다.

| 콘텐츠 유형 | 실행 명령어 | 설명 |
| :--- | :--- | :--- |
| **바이럴 커머스** | `python run_commerce_trend_v1.py` | 실시간 핫템 리뷰 영상 |
| **로또 냥도사** | `python run_lotto_cat_v3.py` | 고양이 스토리텔링 + 로또 |
| **두바이 초콜릿** | `python run_dubai_chocolate_daily.py` | 시리즈물 단독 연재 |

---

## ⚠️ 주의사항 및 환경 설정

-   **API 키 확인:** `.env` 파일에 `GEMINI_API_KEY`와 `YOUTUBE_DATA_API_KEY`가 설정되어 있어야 합니다.
-   **로그 확인:** 모든 작업 내역은 `logs/` 폴더 내의 `.log` 파일과 `.json` 리포트로 기록됩니다.
-   **에셋 저장:** 생성된 영상은 `output_assets/` 폴더에서 확인할 수 있습니다.
